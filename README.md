The `data/` folder contains the necessary datasets for this study:

* Data required to construct the **cooling dehumidification** and **reheat dehumidification** characteristic models.
* Data from the **entire two-month experimental period**, which is used to evaluate the overall predictive performance of the proposed framework.

The `experimental_results.py` script includes the **two-step evaluation** designed to examine the performance of the proposed framework. Specifically, it covers the sections: **4.2 Model Parameter Calibration:**,  **4.3 Comparative analysis of dehumidification characteristics**, and **4.4 Comparative analysis of power consumption**.

The `main.py` script demonstrates the construction of the **cooling dehumidification** and **reheat dehumidification** characteristic models and provides the implementation to derive the **total power consumption** and **latent heat removal** under various operating conditions.
